# 매일 뉴스 정리 프롬프트

## 사용 방법
아래 프롬프트를 AI에 입력하여 날짜별 뉴스를 정리합니다.
파일명 형식: `md/{카테고리명 영어소문자}/YYYY-MM-DD.md`
저장 경로 예시: 
- `md/economy/2026-02-26.md`
- `md/tech/2026-02-26.md`
---

## 프롬프트

```
오늘 날짜: {{YYYY년 MM월 DD일}}
최근 24시간 뉴스와 최신 시장 데이터를 기준으로 아래 형식에 맞춰 뉴스를 정리해줘.

조건:
- 각 카테고리별로 별도의 마크다운 파일을 생성해야 해.
- 각 파일 내용 시작 전에 파일의 저장 경로를 명시하고 마크다운 코드블록(` ```markdown `)으로 내용을 감싸줘.
- 파일의 상단에는 YAML Frontmatter 형식으로 날짜, 카테고리, 그리고 핵심 내용을 요약하는 3줄 요약(`summary`)을 반드시 기재해.
- 루머 또는 커뮤니티성 정보는 [루머] 태그를 붙여.
- 각 뉴스 항목마다 [출처: [출처명](URL)] 형태로 원문 링크를 포함해줘.

---

### [출력 형식 예시]

파일 경로: `md/economy/{{YYYY-MM-DD}}.md`
```markdown

category: "경제"
summary: "미국 연준 금리 동결 시사 및 증시 반등 / 달러 강세 지속 / 코스피 기관 매도세로 소폭 하락"

# {{YYYY년 MM월 DD일}} 경제 뉴스

| 지수 | 현재가 | 전일 대비 | 변동률 |
|------|--------|-----------|--------|
| S&P 500 | | | |
(표 작성...)

## 2. 오늘 시장에 영향을 준 주요 재료
- 항목 1... [출처: [Reuters](URL)]
```

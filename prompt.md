# 매일 뉴스 정리 프롬프트

## 사용 방법
아래 프롬프트를 AI에 입력하여 날짜별 뉴스를 정리합니다.
파일명 형식: `YYYYMMDD_dailynews.md`
저장 경로: `\md\YYYY\MM\YYYYMMDD_dailynews.md`
예시: `\md\2026\02\20260224_dailynews.md`
---

## 프롬프트

```
오늘 날짜: {{YYYY년 MM월 DD일}}
최근 24시간 뉴스와 최신 시장 데이터를 기준으로 아래 형식에 맞춰 뉴스를 정리해줘.

조건:
- 루머 또는 커뮤니티성 정보는 [루머] 태그를 앞에 붙여줘.
- 각 정보 출처의 신뢰도를 [높음 / 중간 / 낮음] 으로 표시해줘.
- 수치 데이터는 전일 대비 변동폭(%, 포인트)을 함께 표시해줘.
- 뉴스 항목마다 출처 URL을 요약 뒤에 표시해줘. 형식: [출처: [출처명](URL)]
- 각 뉴스 항목은 5~10줄 분량의 요약을 포함해줘. 요약 다음 줄에 출처를 표시해줘.
- 각 섹션은 아래 마크다운 형식을 그대로 사용해줘.
```
